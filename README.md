Credit Card Transaction Dashboard

A comprehensive credit card weekly dashboard that provides real-time insights into key performance metrics and trends, enabling stakeholders to monitor and analyze credit card operations effectively. This project leverages SQL for data preparation and Power BI for interactive visualizations.

*Project Overview

This project is designed to track and analyze credit card operations at both transaction and customer levels. It provides stakeholders with actionable insights on credit card performance, usage patterns, and customer behavior, helping drive business decisions.

The dashboard updates weekly and includes key metrics such as spend, revenue, growth, delinquency, and customer segmentation.

*Technology Stack
Database: PostgreSQL

Data Preparation: SQL (joins, aggregations, cleaning)

Visualization & Dashboard: Power BI

Data Sources: Credit card transaction and customer datasets (CSV/Database)
